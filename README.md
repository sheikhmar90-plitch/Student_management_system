Overview

The Student Management System is a Python-based console application designed to efficiently manage student records. It provides a simple menu-driven interface to perform core operations such as adding, viewing, searching, and deleting student data. Each record contains essential information including the student's name, roll number, and marks. This project demonstrates fundamental programming concepts and simulates a basic database management system.

Features
Add new student records
Display all stored students
Search student by roll number
Delete student records
Continuous interactive menu system
Technologies Used
Python 3
Core Python (no external libraries)
Core Concepts Implemented
Lists for data storage
Dictionaries for structured records
Loops (while, for)
Conditional statements (if-else)
User input handling
Basic CRUD operations
How It Works
The program initializes an empty list to store student data
A menu-driven loop allows user interaction
Based on user input:
New student records are added as dictionaries
All records are displayed in structured format
Specific students are searched using roll number
Records are deleted if found
The system continues running until the user chooses to exit
